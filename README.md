# Recon Revenant

Privacy-safe, offline reconnaissance and post-enumeration analysis tool
for Linux and Windows targets.

## Install (pipx)

```bash
pipx install git+https://github.com/<your-username>/recon-revenant.git
```

## Usage

```bash
recon-revenant --nmap nmap.txt --enum linpeas.txti
```

## Privacy

* No network calls
* No telemetry
* No cloud APIs
* Safe for NDA environments
